# CBlog - A simple blogging platform for students
CBlog is a simple blogging platform designed for students to create, read, and manage blog posts. It features user authentication, post creation, editing, and deletion, as well as a comment system.

### Features
- User authentication (registration, login, logout)
- Create, read, update and delete blog posts
- User profile management
- Three types of users: admin, moderator and reader
